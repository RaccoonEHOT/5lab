обработка создания нового пользователя с указанными логином и паролем Про документацию почитай тут.Пример применения:

IDoctorEntry doctor=onRegisterClick(login, password, repPassword); ErrorLabel.Text = "Пользователь " + doctor.Login + " зарегистрирован!";

Аргументы
loginлогин пользователя, не может быть пустым, должен состоять только из цифр,букв и символа _.
passwordпароль, не может совпадать с логином, не может быть менее 10 символов,содержит хотя бы один символ цифры, символ в верхнем регистре
repPasswordповтор пароля, должен повторять password
Возвращает
экземпляр учетных данных пользователя
